Tutorial: first steps using AppInventor

AppInventor is a webbased tool that lets you create apps for Android and iOS using the Scratch IDE. It is easier to start with than JavaFX, but creating more complex applications will require a lot of mouse usage.

You need a Google account for using AppInventor.

Basics

Go to https://ai2.appinventor.mit.edu/ and create an account by logging in using a Google account.

On the Project menu, use "Start new project" to create a project. Give the project a name.

You can now drag userinterface elements from the left onto the phone screen:

  • Add a button
  • Add a label

On the right of the screen, using "Blocks", you can add logic to the screen.

  1. Activate the "Blocks" tab
  2. Click "Button1"
  3. Drag a "When Button1 Click" block onto the canvas.

Now click "Label1", and drag a "set Label1 Text to" block into the click block.

After that, click "Text", and drag the empty string (topmost option) into the "set text to" block. Change the text to something nice (for example "Hello!").

You can now test your app.

  • install the MIT AI2 companion app on your device
  • in menu "Connect", select "AI companion"
  • scan the QR code with the MIT AI2 app
  • after a while the app appears on your phone

Add an image on a new screen

  1. In the designer view, click the button "Add Screen..." and give the new screen a name.
  2. Drag an image onto the canvas.
  3. Upload a random image you like (I used this image of Sealeo)
  4. In the properties of the Image, set the uploaded image as Picture.
  5. Also in the properties, set the Width an Height to "Fill parent", and check ScalePictureToFit.

Select "Screen1" again, and return to the "Blocks" tab.

From the "Control" blocks, drag "open another screen" into the click block. Change the screen name to "Screen2" or whatever you called your second screen.

Now test the app again.
